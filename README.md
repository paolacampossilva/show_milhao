# Descrição
Esse é um trabalho prático realizado na disciplina de Algoritmos e Programação II, componente curricular do 2º período do curso de Ciência da Computação na Universidade Federal de Mato Grosso do Sul.

O trabalho consistia em replicar o funcionamento do real Show do Milhão por meio das instruções providenciadas pelo professor. Pela preferência dos autores foram colocadas funcionalidades adicionais como, por exemplo, a criação e manutenção de um "leaderboard" armazenado em um arquivo binário leaderboard.dat feito para referenciar os "leaderboars" dos originais jogos de "arcade". 

As instruções do trabalho e o relatório final estão ambos disponíveis nesse repositório na forma de arquivos pdf, bem como o arquivo de perguntas.dat necessário para a execução do programa.
